# Website hỗ trợ quản lý thông báo và nhắc lịch trong khách sạn

## 1. Giới thiệu đề tài
Đây là đồ án môn **Công nghệ phần mềm**, đề tài:
**Xây dựng website hỗ trợ quản lý thông báo và nhắc lịch trong khách sạn theo mô hình Agile Scrum**

Hệ thống hỗ trợ quản lý tài khoản nhân viên, quản lý trạng thái phòng, gửi thông báo nội bộ, tạo lịch nhắc check-in/check-out và theo dõi công việc trong khách sạn.

---

## 2. Công nghệ sử dụng
* **Frontend:** ReactJS (Single Page Application)
* **Backend API:** Python (FastAPI) + SQLAlchemy + SQLite
* **Quản lý dự án:** Jira Scrum, Git/GitHub

---

## 3. Khởi chạy dự án (Frontend)

Trong thư mục `frontend-app`, bạn có thể chạy các lệnh sau:

* **`npm start`**: Chạy ứng dụng ở chế độ phát triển tại địa chỉ `http://localhost:3000`.
* **`npm run build`**: Đóng gói dự án để triển khai thực tế.
